One quick step before you start
⚠️
Cannot run from a local file
Firebase requires the app to be served over HTTP — not opened directly as a file. This is a browser security restriction, not a bug.
Choose one of these options:
⚡ Option 1 — VS Code (Easiest)
  1. Install VS Code
  2. Install the Live Server extension
  3. Right-click the HTML file
  4. Click "Open with Live Server"
Opens at http://127.0.0.1:5500
🐍 Option 2 — Python (If installed)
  1. Open a terminal / Command Prompt
  2. Navigate to the file folder:
    cd Downloads
  3. Run:
    python -m http.server 8080
  4. Open http://localhost:8080
🔥 Option 3 — Firebase Hosting
  1. Install Firebase CLI:
    npm install -g firebase-tools
  2. Run firebase init hosting
  3. Place this file in the public folder
  4. Run firebase deploy
🌐 Also Required: Authorize Domain
  1. Go to console.firebase.google.com
  2. Open your project → Authentication
  3. Click Settings → Authorized domains
  4. Add localhost or your domain
Firestore Security Rules — also paste these into your Firebase Console → Firestore → Rules tab before going live. The rules file was included in your project package.
